# marketprism Experiments Source Code:

marketPrism is a secure hybrid sidechain-sharding layer-2 solution for decentralized resource markets.
marketPrism enables effective scaling by accounting for the work models and traffic patterns of these markets, and parallelizing processing by assigning traffic to each sidechain based on its functionality.
